## 要点

GitNexus 是一个完全在浏览器中运行的代码知识图谱生成器。用户拖入 GitHub 仓库链接或 ZIP 文件，即可在客户端生成交互式知识图谱，无需服务器。

内置 Graph RAG Agent，支持基于图谱的代码问答。项目在 GitHub Trending 上获得 16,753 stars，今日新增 1,116 stars。

## 🤖 AI 点评

零服务器架构是亮点——代码分析和图谱构建全在浏览器完成，规避了隐私和部署成本问题。Graph RAG 比传统向量检索更适合代码理解场景，因为代码本身就是图结构（调用关系、依赖树）。实际效果取决于客户端性能和大型仓库的处理能力。
