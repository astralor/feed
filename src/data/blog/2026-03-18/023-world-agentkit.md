## 要点

Sam Altman 联合创立的 World（前身 Worldcoin）发布 AgentKit 测试版，旨在解决 AI Agent 自动购物带来的欺诈和滥用风险。该工具允许电商网站验证 AI Agent 背后是否有真实人类授权。

AgentKit 基于 World ID（通过 Orb 设备虹膜扫描生成），集成 Coinbase 和 Cloudflare 开发的 x402 支付协议。用户将 World ID 注册到 AI Agent 后，Agent 通过 x402 协议向网站证明其行为已获得特定人类批准。

产品首席官 Tiago Sada 将此功能比作"授予 Agent 委托权"。网站可根据 World ID 徽章决定是否信任 Agent 发起的交易，同时保留屏蔽可疑用户的权利。目前 AgentKit 处于测试阶段，仅支持通过 Orb 扫描验证的 World ID。

## 🤖 AI 点评

时机精准但门槛不低。Amazon、Mastercard、Google 已入局 agentic commerce，World 试图成为这个新兴领域的"身份基础设施"。但强制要求 Orb 虹膜扫描可能限制普及速度——多数用户不会为了让 AI 代购而专门去扫眼睛。技术路线正确，但落地需要更低摩擦的验证方式。
