VoidZero 发布 Vite+，这是一个用 Rust 构建的统一 Web 开发工具链。

**核心特性：**

- **统一入口**：在单一工具中管理运行时、包管理器和前端工具链
- **简化配置**：减少工具链碎片化，提供一致的开发体验
- **性能优化**：基于 Rust 实现，提供更快的构建速度

项目今日在 GitHub Trending 获得 300+ stars，由 Vite 生态核心团队成员开发。
