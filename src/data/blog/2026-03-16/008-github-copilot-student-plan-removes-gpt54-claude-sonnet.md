GitHub 宣布调整免费学生 Copilot 计划，将多个高成本模型从免费额度中移除，引发开发者社区讨论。

## 被移除的模型

- GPT-5.4（Input $2.50/1M，Output $15/1M）
- Claude Sonnet 4.6（Input $3/1M，Output $15/1M）
- Claude Opus 4.6（Input $5/1M，Output $25/1M）

## 仍可免费使用的模型

- Claude 4.5 Haiku（Input $1/1M，Output $5/1M）
- Gemini 3.1 Pro（Input $2/1M，Output $12/1M）
- GPT-5.3 Codex（Input $1.75/1M，Output $14/1M）

## 解读

GitHub 官方表示「这一调整是为了让 Copilot 对全球数百万学生持续可用、保持可及性」。但实质是高端模型的推理成本已不可持续免费提供。

这一变化折射出 AI 免费增值模式的内在张力：当最好的模型越来越贵，「免费体验」只能提供降档版本。
