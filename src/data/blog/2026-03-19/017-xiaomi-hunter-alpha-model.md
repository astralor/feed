> 评分依据：行业级反转事件，小米以非传统玩家身份推出 1T 参数 Agent 旗舰模型

## 要点

3 月 11 日，一个名为 Hunter Alpha 的匿名模型悄然登陆 OpenRouter，标注为 1 万亿参数、100 万 token 上下文窗口。由于参数规格和知识截止时间（2025年5月）与市场预期的 DeepSeek V4 高度吻合，整个 AI 社区一周内都在猜测这是 DeepSeek 的秘密测试。3 月 18 日，小米 AI 模型团队 MiMo 认领了该模型——它是 MiMo-V2-Pro 的早期内部测试版。

MiMo 由前 DeepSeek 研究员罗福莉领导，定位为 Agent 专用「大脑」，设计目标是让用户以更少的人类提示和监督完成复杂任务。模型上线后迅速突破 1 万亿 token 总使用量，登顶 OpenRouter 排行榜。小米宣布将与 OpenClaw 等五大 Agent 框架合作，面向全球开发者提供一周免费访问。小米港股当日上涨 5.8%。

## 🤖 AI 点评

这个故事最有意思的不是「谁做的」，而是「为什么没人猜到是小米」。DeepSeek 的光环太强，以至于当一个规格相当的匿名模型出现时，所有人的第一反应都是「一定是 DeepSeek V4」。但罗福莉从 DeepSeek 出来后选择加入手机公司做 Agent 模型，这个选择本身就释放了信号：大模型的战场正在从「谁能训练最大的基座模型」转向「谁能做出最好用的 Agent 专用模型」。1T 参数 + 100万上下文 + 免费开放 + 绑定 Agent 框架——小米打的不是模型能力竞赛，是生态渗透战。
