## 要点

Google 正在重组 Project Mariner 团队——这是其能够操控 Chrome 浏览器、替用户完成在线任务的 AI Agent 研究原型。据 WIRED 了解，近几个月来部分 Google Labs 研究人员已转向更高优先级项目。

Google 发言人确认了这一变动，表示 Project Mariner 开发的计算机操控能力将被整合进公司未来的 Agent 战略，部分能力已并入近期发布的 Gemini Agent。值得注意的是，去年 I/O 大会上 Sundar Pichai 曾亲自推介 Project Mariner，当时浏览器 Agent 被视为行业下一个大赌注。

现实却给了不同的答案：Perplexity 的 Comet 浏览器 Agent 周活跃用户仅 280 万，远低于行业预期。与此同时，OpenClaw 席卷开发者社区，黄仁勋在 GTC 上称其为「Agent 计算机的新操作系统」，声称每家公司都需要 OpenClaw 战略。

## 🤖 AI 点评

Project Mariner 的命运折射出一个关键问题：浏览器 Agent 和系统级 Agent 究竟哪条路通向未来。Google 的答案很明确——放弃专用浏览器 Agent，转向通用 Gemini Agent，这本质上是从「让 AI 用浏览器」到「让 AI 直接操作系统」的范式转移。Perplexity Comet 的 280 万周活说明，浏览器 Agent 在消费端的需求被高估了——开发者真正想要的不是自动化网页点击，而是直接掌控整个计算环境的能力。
