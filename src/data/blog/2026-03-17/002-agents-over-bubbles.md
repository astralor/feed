## 要点

Ben Thompson 在 NVIDIA GTC 2026 前夕发表长文，论证当前 AI 投资热潮**不是泡沫**，核心论据：

**LLM 三次范式转变：**
1. **ChatGPT（2022.11）** — 让 LLM 可用，但会犯错、需要人类主动引导
2. **o1（2024.09）** — 推理模型自我验证，减少人类干预，但推理 token 消耗暴增
3. **Opus 4.5 + Agent Harness（2025.12）** — Agent 自主调用模型+工具验证结果，算力需求再次跳跃

**为什么不是泡沫：**
- **算力需求爆炸式增长** — Agent 需要多次调用推理模型 + CPU 执行工具，每个范式转变都让算力需求翻倍
- **Agency 门槛降低** — 不需要所有人都用 AI，只需少数有 agency 的人通过 Agent 控制多个任务，就能产生巨大算力需求
- **企业经济驱动** — 企业不只是用 AI 提效，而是用 Agent 替代组织中的"人类齿轮"，既降本又增效，经济激励无法抗拒

**价值链重构：**
- **模型不再是商品** — Agent 需要模型与 harness 深度整合，Anthropic 和 OpenAI 成为整合点，利润不会被商品化
- **Microsoft 的转向** — 从"模型无关"战略转向 Copilot Cowork（基于 Claude），承认 Agent 时代无法抽象掉模型
- **Apple 的风险** — 只授权 Gemini 而自建 harness，可能无法交付有竞争力的 Agent 产品

**裁员潮预警：**
Thompson 预测即将到来的 AI 驱动裁员潮不只是"疫情过度招聘"的修正，而是企业发现**后 AI 时代的合理规模远小于前 AI 时代**，会"宁可多裁也不少裁"，用 Agent 重建规模。

## 🤖 AI 点评

这篇文章最有价值的地方在于**用经济学逻辑而非技术炒作来论证 AI 投资的合理性**。Thompson 不是说"AI 很厉害所以值得投资"，而是说"Agent 改变了算力需求曲线、降低了 agency 门槛、重构了企业经济学，所以超大规模算力投资是供不应求而非投机"。

特别值得关注的是**价值链整合论断**：如果 Agent 确实需要模型与 harness 深度整合（而非模块化组装），那么 Anthropic/OpenAI 的护城河比想象中更深，Microsoft 的"模型无关"战略已经失败，Apple 的"授权模型+自建 harness"路线也岌岌可危。这对整个 AI 产业格局有深远影响。

文末的自我解构也很有意思：Thompson 说"当我宣称不是泡沫时，可能恰恰证明是泡沫"——这种自我怀疑反而增加了论证的可信度。
