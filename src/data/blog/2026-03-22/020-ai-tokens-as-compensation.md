## 要点

Jensen Huang 在 GTC 2026 上提出，工程师应获得约等于基础薪资一半的 AI token 预算——顶尖工程师可能每年消耗 $250,000 的推理算力。他将此定位为招聘工具，并预测将成为硅谷标配。

Theory Ventures 合伙人 Tomasz Tunguz 早在 2 月中旬便提出「推理即薪酬」概念，用 Levels.fyi 数据推算：年薪 $375K 的工程师加上 $100K token 预算后，「每花五美元中就有一美元是算力」。这一趋势的催化剂是 Agentic AI 的爆发——OpenClaw 等工具让开发者可以 7×24 不间断运行 Agent 完成编码、研究和自动化任务，token 消耗量因此指数级增长。

但文章也指出了反面论据：token 预算本质上是对公司的锁定，工程师若离职则全部归零；且随着模型价格持续下降，今天的「慷慨」可能很快变成鸡肋。核心问题是——这到底是下一代 signing bonus，还是公司把本应承担的基础设施成本转嫁给了员工？

## 🤖 AI 点评

「AI token 作为薪酬」这个概念比表面看起来更深。它反映了一个结构性变化：当 Agent 成为开发者的核心生产力工具时，算力分配就是能力分配。公司给 token 不只是给福利，是在说「我们希望你用 Agent 干更多活」。从这个角度看，$250K 的 token 预算更像是对 AGI 时代工程师角色重新定价的一次试探——前提是这些 token 真的能产出等值的代码。
