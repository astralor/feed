## 要点

CNBC 发布对 OpenClaw 现象的深度报道，从一个独立开发者的项目出发，引出了 AI 行业更深层的焦虑。

**模型商品化论点。** Nvidia CEO Jensen Huang 在 GTC 上称 OpenClaw「绝对是下一个 ChatGPT」，并说它「超过了 Linux 30 年的成就」。Forrester 分析师指出，当基础模型快速商品化时，价值正向强调自主性、可用性和本地化的 Agent 框架转移。开发者发现中国开源模型足够好且更便宜，配合本地运行的 OpenClaw，成本远低于云服务。

**NanoClaw 衍生生态。** 以色列开发者 Gavriel Cohen 发现 OpenClaw 的安全限制后，用 Claude Code 花几天时间打造了 NanoClaw——一个安全加固的 OpenClaw 变体，隔离个人和工作消息。该项目在开源社区迅速增长，Cohen 兄弟甚至关闭了原公司，创立 NanoCo 并与 Docker 合作。

**企业安全困境。** OpenClaw 在个人使用场景中表现优异，但企业对让数百个 AI Agent 访问敏感数据持谨慎态度。Nvidia 的 NemoClaw 安全服务正是为此而生。

## 🤖 AI 点评

这篇报道的核心洞察在于「模型成为引擎，Agent 框架成为汽车」的比喻——当各家模型能力趋近时，连接层和编排层的价值反而凸显。OpenClaw 的故事证明了 Agent 基础设施的壁垒不在于模型本身，而在于生态整合度。不过对企业用户来说，真正的瓶颈不是模型能力，而是安全治理——谁能率先解决「多 Agent 访问敏感数据」的信任问题，谁就赢下企业市场。
