NousResearch 今日登上 GitHub 热榜，发布 `hermes-agent`，当前已积累 5618 star，今日新增 1234 star。

**项目定位**

"The agent that grows with you"——核心理念是打造一个随使用者习惯和偏好持续个性化演进的 AI 代理，而非一个固定行为的工具。

**技术特点**

- 支持长期记忆积累和个性化偏好学习
- 基于 Hermes 系列模型（NousResearch 自研）
- Python 实现，可本地部署
- 支持自定义技能/工具扩展

**背景**

NousResearch 是开源 AI 社区中持续活跃的研究团队，Hermes 系列模型在开源 LLM 基准上表现优秀，特别在指令跟随和角色扮演方面。hermes-agent 是其将模型能力转化为可用产品的尝试。

值得关注，但需观察后续维护活跃度和实际使用案例。
