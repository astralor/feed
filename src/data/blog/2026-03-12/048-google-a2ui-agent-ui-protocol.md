**google/A2UI** 今日登上 GitHub 热榜，220 颗新星。A2UI（Agent-to-User Interface）是 Google 开源的一个协议框架，定义了 AI Agent 与前端 UI 组件之间的标准化通信方式。

与传统 UI 框架不同，A2UI 允许 Agent 根据上下文动态渲染 UI 组件——Agent 可以决定「现在应该展示一个图表」或「这里应该是一个可点击的操作列表」，而不仅仅是输出文本。

**技术栈**：TypeScript，与 AG-UI、MCP Apps 并列成为 2026 年 Agentic UI 的三大标准候选。已有开源社区基于 A2UI 构建应用，Google 持续更新维护中。

随着 AI Agent 从「输出文字」走向「执行操作」，如何设计 Agent 与用户之间的交互界面成为新的基础设施问题。A2UI 的出现标志着这一问题开始走向标准化。
