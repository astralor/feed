Datadog 于 3 月 9 日正式宣布 **MCP Server 进入 GA（正式可用）**，这是 AI 可观测性领域的重要里程碑。

**核心价值**：通过 Model Context Protocol (MCP)，AI Agent 和 IDE 插件可以安全、实时地访问 Datadog 统一可观测性数据——包括日志、指标、追踪、告警。开发者在调试问题时，AI 助手能直接查询生产环境的真实数据，而非依赖历史快照。

**使用场景**：AI 编程 Agent 自动排查线上故障时，可调用 MCP Server 获取实时指标；IDE 中的 AI 助手可以在代码编辑的同时查看相关服务的运行状态；Agent 工作流可以基于可观测性数据做自动化运维决策。

Datadog MCP Server 的 GA 代表着可观测性平台正在深度融入 AI Agent 工具链，「AI Agent + 实时基础设施数据」的组合将使自动化运维更可靠。继 Galileo 开源 Agent 可观测控制平面之后，这一领域热度持续升温。
