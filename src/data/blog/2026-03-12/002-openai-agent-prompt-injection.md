OpenAI 安全团队发布了关于 AI Agent 安全性的重要研究，聚焦于如何设计 Agent 系统来抵御 prompt injection 攻击。随着 Agent 越来越多地被用于自主执行任务、访问外部数据源和工具，prompt injection 成为关键安全挑战。

该研究提出了多层防御策略，对于所有构建 Agent 系统的团队都有参考价值。
