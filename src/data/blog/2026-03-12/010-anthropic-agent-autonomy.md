Anthropic 社会影响研究团队发布了 AI Agent 自主性的实践度量方法。随着 Agent 系统越来越多地被部署到生产环境，如何量化和控制 Agent 的自主程度成为关键安全问题。

该研究提出了一个实用框架，可以帮助开发者和部署者理解和评估 Agent 系统在实际使用中的自主行为边界。
