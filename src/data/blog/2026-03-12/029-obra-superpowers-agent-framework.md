`obra/superpowers` 今日登上 GitHub 热榜，这个拥有 78K star 的仓库是一套结合"Agentic 技能框架"与"软件开发方法论"的开源项目。

**项目定位**

- 为 AI 代理（Claude、Codex 等）定义可复用的"技能"（Skills）
- 提供一套软件开发方法论，让 AI 代理能够更可靠地协作完成复杂任务
- 用 Shell 实现，跨平台兼容性强

**为何获得关注**

随着 Claude Code、Codex 等编程代理的普及，如何让代理保持上下文、执行复杂多步任务、避免"幻觉式操作"成为核心痛点。`superpowers` 提供了一种结构化的解决思路，类似于给代理装上"行为规范"。

**社区反应**

HN 和 X 上有大量用户分享基于此框架的工程实践，特别是在"多代理协作"和"长时任务规划"方面的应用案例。

对于深度使用 AI 编程代理的开发者，值得一读。
