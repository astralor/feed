OpenAI 推出 AgentKit，这是基于此前发布的 Responses API 和 Agents SDK 构建的端到端 Agent 开发框架。

官方案例展示了实际应用效果：Klarna 用其构建的客服 Agent 处理了三分之二的工单，Clay 的销售 Agent 实现了 10 倍增长。AgentKit 旨在让开发者更便捷地构建深度研究、客户支持等 Agentic 工作流。
