JetBrains 宣布 **Junie CLI** 正式进入 Beta，将其智能编程 Agent 从 IDE 扩展升级为完全独立的命令行工具。

**核心特性**：
- **LLM 无关**：不绑定特定模型，可灵活切换底层 AI
- **全场景覆盖**：终端、任意 IDE、CI/CD 流水线、GitHub/GitLab 均可接入
- **深度代码库理解**：以项目上下文为基础，而非孤立执行指令

**配套发布**：JetBrains 同步推出 **Air**（公开预览版），一款为 Agent 时代重新设计的 IDE，支持 Codex、Claude Agent、Gemini CLI 和 Junie 等多个 AI Agent 开箱即用，并允许跨项目自由切换 Agent。

**行业意义**：JetBrains 此举是传统 IDE 厂商回应 AI 原生开发工具冲击的最重要案例。在 Cursor、Claude Code、Codex 持续攻占开发者心智的背景下，Junie CLI 为 JetBrains 的 2600 万用户提供了一条不脱离现有工具链的 Agent 升级路径。
