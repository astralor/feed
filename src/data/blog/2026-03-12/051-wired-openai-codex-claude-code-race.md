Wired 发布深度独家报道，揭示编程 Agent 赛道正成为 OpenAI 和 Anthropic 之间最激烈的战场。

**核心数据**：Claude Code 已占 Anthropic 业务近五分之一，折合年化营收超 25 亿美元。这一数字让 OpenAI 高度警觉，内部正全力加速 Codex 的研发节奏。

**竞争格局**：
- Anthropic 通过 Claude Code 建立了强大的开发者生态粘性，开发者几乎把 Claude Code 当成"键盘"来用
- OpenAI 的 Codex 已被部分员工用于夜间跑任务，次日检查结果——但产品成熟度和开发者采纳率仍落后
- OpenAI 正同时推进更通用的计算机操控 Agent，试图从另一维度反包围

**行业意义**：编程 Agent 不再是实验性功能，而是 AI 公司的核心商业支柱。谁能赢得开发者的工作流，谁就赢得了最高频、最高价值的 AI 使用场景。
