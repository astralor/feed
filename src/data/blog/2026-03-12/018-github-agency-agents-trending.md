`msitarzewski/agency-agents` 今日在 GitHub Trending 榜首，单日涨星 **6,167**，总星数超过 30,000。

## 是什么

一个"完整 AI 代理机构"——仓库里内置了多种专业化 Agent：

- **Frontend Wizard** — 前端开发代理
- **Reddit Community Ninja** — 社区运营代理  
- **Whimsy Injector** — 创意注入代理
- **Reality Checker** — 事实核查代理
- ...等十余种专业角色

每个 Agent 都有自己的「人格」、处理流程和标准交付物格式，可以组合协作。

## 为什么突然爆火

这类"Agent 角色扮演框架"的爆火有其结构性原因：Claude/GPT 在角色一致性上的大幅提升让 Agent 的专业化成为可能，开发者开始寻找开箱即用的 Agent 团队模板。
