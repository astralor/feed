OpenAI 于 3 月 7 日正式宣布 **Codex for Open Source** 计划，旨在支持承担重要开源生态工作的核心维护者。

**计划包含：**
- **6 个月免费 ChatGPT Pro**（含 Codex），可用于日常编码、triage、代码审查、维护工作流
- **Codex Security 条件访问**，针对需要深度安全扫描的仓库
- **Codex Open Source Fund API 积分**，适用于将 Codex 集成到 PR review、维护自动化、发布工作流等核心 OSS 工作的项目

**申请条件：** OpenAI 将根据仓库使用量、生态系统影响力等因素审核申请，重点支持在开源生态中扮演关键角色的公共仓库维护者。

这一计划背后有一个现实背景：AI 生成的 PR 大量涌入开源项目，人工维护者的负担越来越重。OpenAI 这步棋，是在把 Codex 定位为维护者的协作伙伴，而不仅仅是贡献者的工具。
