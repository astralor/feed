OpenAI 宣布将收购 AI 安全评估平台 Promptfoo，由 Ian Webster 和 Michael D'Angelo 创立，目前被超过 25% 的财富 500 强企业使用。

**收购后的整合计划：**

- Promptfoo 将直接整合进 OpenAI Frontier（企业 AI 协作者平台）
- 在平台内内置安全测试和 Red Team 功能，自动检测 prompt injection、越狱、数据泄露等风险
- Agent 开发工作流中嵌入安全评估能力，让安全测试成为开发过程的一部分
- 提供内置的可追溯性与合规报告，帮助企业满足治理要求

**背景价值：**

随着企业加速部署 AI Agent 进入真实业务流程，如何在部署前系统性发现 Agent 行为风险成为核心诉求。Promptfoo 开源工具已有大量社区积累（GitHub 上的 CLI 和库），此次收购兼顾了开源延续和企业能力升级，是 OpenAI 强化 B2B 安全基础设施的关键一步。
