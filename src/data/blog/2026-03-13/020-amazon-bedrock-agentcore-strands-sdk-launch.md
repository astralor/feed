在 NVIDIA GTC 2026 召开前夕，AWS 悄悄发布了一套完整的 Agent 基础设施栈，这很可能是本周被低估的重要发布。

## 两个产品，一个目标

**Amazon Bedrock AgentCore** 是企业级托管 Agent 运行时，提供安全隔离、可观测性、持久化 Memory、工具网关等生产必备基础设施，完全模型无关（支持 OpenAI、Claude、Gemini、Llama 等）。

**Strands Agents** 是开源轻量 SDK，「用几行代码构建 Agent，无需手写编排逻辑」。框架兼容性涵盖 LangGraph、Microsoft AutoGen、OpenAI Agents SDK、CrewAI、Google ADK 和 Strands 自身。

## Strands Labs

AWS 同步开放了 **Strands Labs** — 一个实验性 GitHub 组织，专注 Agent 下一代方向的探索：从 Agentic Robotics 到让现有应用「更具 Agent 性」。Clare Liguori（AWS 高级首席工程师）定义它为「Agent 开发下一代想法的游乐场」。

## 为何值得关注

AWS 的这套组合拳表明：AI 基础设施的竞争已从**模型层**转移到**运行时和框架层**。谁先建立 Agent 运行时标准，谁就可能在下一轮云计算竞争中占据有利位置。
