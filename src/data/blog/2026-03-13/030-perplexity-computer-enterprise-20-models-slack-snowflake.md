Perplexity 在 **Ask 2026 开发者大会**上正式发布 **Computer for Enterprise**，将此前面向个人的 AI Agent 产品推向企业市场，直接与 Microsoft Copilot 和 Salesforce Einstein 竞争。

## 产品能力

- **多模型编排**：同时调度 20 个 AI 模型协同完成复杂任务
- **Slack 原生集成**：支持在 Slack 工作流中直接触发 AI Agent
- **Snowflake 连接器**：企业数据直接接入，无需手动导出
- **统一账单**：单一 API 密钥访问来自 6 家提供商（OpenAI、Anthropic、Google、xAI、NVIDIA、Perplexity）的模型

## 竞争定位

Perplexity CTO Denis Yarats 明确表示，**Agent API 的目标是替代原本需要拼接的完整技术栈**：模型路由器 + 搜索层 + 嵌入服务 + 沙箱 + 监控——现在都用一个端点搞定。

同时，Yarats 对 MCP（Model Context Protocol）提出批评，认为其上下文开销和认证摩擦在规模化场景下不实用。

## 影响

这是 AI Agent 平台层竞争的重要一步，标志着 Perplexity 从「搜索引擎替代品」向「企业 AI 操作系统」的战略转型。
