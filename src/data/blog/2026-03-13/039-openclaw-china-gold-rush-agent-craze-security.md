**MIT Technology Review** 深度报道了 OpenClaw AI Agent 在中国引发的「淘金热」。这个开源 AI 系统在中国被用户昵称为「**龙虾**」（因其 Logo 设计），正以类似 DeepSeek 时刻的速度席卷从极客到企业的各个层面。

## 现象

- 中国软件工程师大量基于 OpenClaw 生态构建衍生工具，包括可视化进度的「桌面小工人」、语音交互插件等
- CNBC 报道：MiniMax 和 Kimi 等中国模型受益于这波热潮，付费模型需求显著提升
- 企业客户加速部署 AI Agent 流程，AI Agent 采用率指数级增长

## 政府介入

然而，2026 年 3 月，中国政府开始**限制国家机构和国有企业使用 OpenClaw**，理由是数据安全顾虑。这一政策与早期对 ChatGPT 的处理方式如出一辙：民间使用热情高涨，官方同步收紧管控。

## 更大含义

这一现象从侧面印证了 AI Agent 作为「个人/企业操作系统」的定位正在被市场验证。OpenClaw 能在中国掀起如此浪潮，说明 AI Agent 范式的吸引力已超越技术极客圈，成为真正的大众产品。
