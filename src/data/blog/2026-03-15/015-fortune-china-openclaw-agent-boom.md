Fortune 深度报道中国 AI Agent 热潮。「养虾」（raise a lobster）这个网络梗描述的是用户把 OpenClaw 当宠物养的现象——来源于平台吉祥物小龙虾——正成为 2026 年中国科技圈最热话题之一。

几乎所有中国主要 AI 实验室都在争相兼容 OpenClaw 生态：
- **月之暗面 Kimi 2.5** — 上下文窗口再度扩展，多模态能力增强
- **Minimax M2.5** — 多模态旗舰模型全面升级
- **智谱 GLM-5** — 已在 0G Compute 平台上线
- **字节跳动 Seedance 2.0** — 视频生成模型在 2026 年春晚爆红

文章分析，中国 Big Tech 公司（腾讯、阿里等）正借助 Agent 浪潮弯道超车 OpenAI 和 Anthropic。相对于封闭的 API 路线，中国实验室更倾向于开源策略，这与 OpenClaw 的 self-hosted 特性完美契合。政府政策也在积极推动 AI Agent 在政务、制造、医疗等领域的大规模采用。
