dimensionalOS/dimos（Dimensional Framework）今日登上 GitHub Trending，主打为具身 AI 和机器人 Agent 设计的统一开发框架。与大多数面向软件 Agent 的框架不同，dimos 专注于物理世界中需要感知-规划-执行循环的应用场景。

框架以 Python 构建，当前已有 933 星，72 颗今日新增。背后团队 dimensionalOS 的目标是让开发者能像搭积木一样组合具身 AI 系统。

**框架特性：**
- 统一感知（视觉/传感器）和语言模型接口
- 支持多模态输入的任务规划
- 内置执行器抽象，适配机器人控制 API
- 支持长期记忆和环境状态维护

随着物理 AI（Physical AI）成为 NVIDIA GTC 2026 的核心议题，dimos 这类框架的出现恰逢其时。
