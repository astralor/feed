纽约时报长篇调查报道深入探讨 AI 对软件工程师职业的颠覆。文章聚焦在 Claude Code 等编程 AI 的帮助下，初创公司正以远少于以往的工程师数量构建复杂产品。

记者走访了多家使用 AI 辅助编程的公司，发现一种新型工作模式：工程师更多承担「架构师」和「AI 指挥者」角色，而非逐行写代码。部分公司甚至以 2 名工程师完成原本需要 10-15 人团队的工作。

**核心发现：**
- Claude Code 正在大量取代初级程序员的日常工作
- 需求正向「能与 AI 协作」的高级工程师转移
- 编程教育模式面临根本性变革
- 就业市场两极分化：顶尖工程师薪酬上涨，初级职位正在萎缩

文章没有给出简单乐观或悲观的结论，而是呈现了这场技术变革中真实存在的焦虑与机遇。
