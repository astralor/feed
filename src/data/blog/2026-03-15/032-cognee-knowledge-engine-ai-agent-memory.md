`topoteretes/cognee` 是一个专为 AI Agent 设计的知识引擎，登上 GitHub Trending。

## 核心能力

- **统一记忆层**：将向量搜索、知识图谱和结构化存储融合为单一 API
- **极简集成**：仅需 6 行代码即可为现有 Agent 添加持久记忆
- **时序推理**：支持「昨天发生了什么」类的时间感知查询
- **跨 Agent 共享**：多个 Agent 可共享同一知识库

## 使用场景

- 长对话 Agent 记忆管理
- 企业知识库 RAG 增强
- 多 Agent 协作中的共享上下文

与 MemGPT、LangMem 等方案相比，cognee 更强调知识图谱结构，适合需要关系推理的应用场景。
