NVIDIA 预计将在本周 GTC 2026 大会（3 月 16 日开幕）上正式发布 **NemoClaw**，一个专为企业级 AI Agent 工作流设计的开源平台。据 The New Stack 消息，NVIDIA 已与 Salesforce、Google 和多家云厂商洽谈集成事宜，将 NemoClaw 定位为"企业部署 AI Agent 的安全底座"。

NemoClaw 主打三大能力：

- **隔离执行环境**：为每个 Agent 提供独立沙箱，防止跨 Agent 数据泄漏，满足金融、医疗等监管行业的合规要求
- **多 Agent 编排**：内置任务路由、结果聚合和失败重试机制，支持复杂的 Agent 协作拓扑
- **与 NVIDIA 推理栈整合**：无缝对接 TensorRT-LLM、NIM Microservices，实现本地或云端低延迟推理

此次发布与 Jensen Huang 在 GTC 主题演讲中反复提及的 "Physical AI + Agentic AI" 双轮战略高度契合。NemoClaw 若成熟，将进一步巩固 NVIDIA 在 AI 价值链中从芯片到平台的垂直整合布局。
