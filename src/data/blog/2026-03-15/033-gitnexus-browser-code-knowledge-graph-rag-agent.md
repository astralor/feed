`abhigyanpatwari/GitNexus` 今日新增 450 颗星，定位「零服务器代码智能引擎」。

## 工作原理

1. 用户拖入 GitHub 仓库链接或 ZIP 文件
2. 浏览器端自动解析代码，构建函数调用图、模块依赖图
3. 内置 Graph RAG Agent，可用自然语言提问：「这个函数在哪里被调用？」「这个模块的依赖链是什么？」

## 技术亮点

- **纯客户端**：所有处理在浏览器端完成，无隐私泄露风险
- **交互式可视化**：知识图谱可缩放探索
- **Graph RAG**：图结构感知检索，比普通向量 RAG 更适合代码关系查询

## 适合场景

快速上手新开源项目、代码审查前的结构预览、文档缺失时的代码探索。
